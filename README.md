Pakistan E-commerce Data
This dataset contains transactional data from an ecommerce platform operating in Pakistan. It includes details on individual orders with columns capturing key aspects of each transaction.
Key Details:
- Order & Item Info: item_id, status (complete, canceled, refund, received), qty_ordered.
- Financials: price, TOTAL price, pricing category (LOWER, STANDARD, UPPER, Premium).
- Categories & Payments: category_name_1 (like Mobiles, Beauty & Grooming), payment_method (COD, Easypay, etc.).
- Customer & Timing: Customer Since month, Order year, M-Year-order.
